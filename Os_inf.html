Recursos

CADASTRO DE SERVIÇO.

Cadastre seus serviços para adicionalos a um orçamento ou venda, o app não permite nomes iguais.

CADASTRO DE EMPRESA/ AUTÔNOMO

Cadastre sua empresa ou seu dados de autônomo para que os dados sejam adicionados aos dados da Ordem de serviço ou do orçamento.

SELEÇÃO DE LOGOTIPO

Selecione uma imagem para mostrar na tela inicial do app e no arquivo de orçamento e OS.

LISTA DE SERVIÇO

Visualise facilmente sua lista de serviço para editala.

CADASTRO DE ORÇAMENTO

Crie ou edite orçamento e ao final envie para seu cliente por Whatsapp.

CADASTRO DE CLIENTE.

Cadastre seus clientes para gerencialos e criar orçamento e ordem de serviço.

TELA DE RESTAURAÇÃO DOS DADOS

Copie o arquivo do banco para a pasta indicada e realize a restauração para não ter que recadastrar todos os dados novamente.

TELA DE BACKUP DOS DADOS.

Faça uma copia dos dados cadastrados usando um pc conectado ao smartphone para quando precisar resetar o celular ou trocar.

TELA INICIAL

Apresenta uma logo padrão que pode ser alterada pela logo da sua empresa.

TELA MENU PRINCIPAL

Menu com icones personalizados para melhor interpretação.

LISTA DE CLIENTE

Tela de lista de cliente para gerenciar ou editar.

LISTA DE ORÇAMENTO

Lista de visualização e edição dos orçamentos.

TELA DE OPÇÕES PARA OR E OS

Tela de opções para editar, excluir ou visualizar em pdf.

ORÇAMENTO OU OS EM PDF.

Aquivo em PDF, que pode ser compartilhado para o cliente.

TELA DE SELEÇÃO DE DATA DA OR/OS.

Selecione uma data diferente para um orçamento atrazado.

TELA DE SELEÇÃO DE CLIENTE.

Tela de seleção de cliente para adicionar ao orçamento.
